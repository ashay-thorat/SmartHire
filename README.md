# SmartHire 💼

SmartHire is an AI-powered job platform that analyzes a candidate's resume, generates a resume score, and recommends relevant jobs based on the skills and content found in the resume.

## 🚀 Features
- Resume Upload and Analysis
- Resume Scoring System
- AI-based Resume Evaluation
- Job Search Based on Resume Skills
- User Authentication (Login / Register)
- Job Listings and Recommendations
- Responsive User Interface

## 🧠 How It Works
1. The user uploads their resume.
2. The system extracts and analyzes resume content.
3. A resume score is generated based on skills and structure.
4. Relevant job opportunities are suggested based on resume skills.

## ⚙️ Installation
Clone the repository
git clone https://github.com/yourusername/SmartHire.git


Navigate to project folder


cd SmartHire


Install frontend dependencies


cd client
npm install
npm run dev


Install backend dependencies


cd server
npm install
npm start


## 🌐 Environment Variables

Create a `.env` file inside the server folder and add:


PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
